$$\log_{\frac{1}{7}}(7 - x) = -2$$

$$49 = 7 - x$$

\section{№ \textbf{77380}}

$$\log_5(x^2 + 2x) = \log_5(x^2 + 10)$$
$$x^2 + 2x = x^2 + 10$$
$$2x = 10$$
$$x = 5$$

Ответ: \textbf{5}

\section{№ \textbf{315120}}

$$\log_{8}2^{8x - 4} = 4$$
$$8^4 = 2^{8x - 4}$$
$$2^12 = 2^{8x - 4}$$
$$12 = 8x - 4$$
$$16 = 8x$$
$$x = 2$$

Ответ: \textbf{2}

\section{№ \textbf{315535}}

$$8^2 = 5x - 3$$
$$64 = 5x - 3$$
$$67 = 5x$$
$$x = \frac{67}{5} = 13.4$$
